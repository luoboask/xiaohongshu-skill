# 配图文字密度参考

## 核心原则

**每张卡片3-4行文字，AI prompt 不超过2句。**

信息图是要人看的，不是说明书。留白比堆字重要。

---

## 行业拆解密度标准

以 `day03-会计.md` 的配图 prompt 为准。

### 配图1（行业结构）密度
- 金字塔5层，每层：`职位名 占比 收入范围`（一行）
- 细分领域：横向堆叠条，每段 `类别名 占比`
- 人群画像：3个饼图，每个饼图4个切片，标注 `类别 占比`
- 底部一句话总结

### 配图2（赚钱逻辑+痛点）密度
- 收入公式：1行公式 + 4个注解（每个不超过1行）
- 收入来源：横向堆叠条
- 定价权：3层，每层 `级别-定价权(一句话原因)`
- 工作时间成本：3级，每级1行
- 4个痛点：每个2行 `具体表现 / 严重程度`

### 配图3（AI重构收益）密度
- 新收入公式：1行公式 + 1行解释 + 3个子公式（每个1行）
- 收入结构饼图：6个切片
- 3大杠杆：每个 `标题 + 1-2行案例`
- 底部badge：`从X → 从卖A → 卖B`

### 配图4（AI实操）密度
- 每张卡片3部分：步骤（左）+ 效果（右）+ 底部条
- 步骤用箭头连接，短语形式
- 效果1-2行
- 底部：`工具 + 耗时 + 效果`
- 底部总结box：1行

## 轻创业密度标准

以 `day14-轻创业后备箱咖啡.md` 和 `day34-陪诊服务.md` 的配图 prompt 为准。

### 黄金样本

### 样本A：day14-轻创业后备箱咖啡（信息图密度标准）
- 配图1~3的卡片密度、文字量、短语长度以此为标准
- 配图4 Card 1 的prompt风格（"辞职做咖啡的第30天 / 每天早起占位置"）
- 正文叙事角度：观察者视角（"我盯了它一个季度"）
- 时间线结构：第一个月→第二个月→第三个月（适合这个项目，因为生意确实在按月变化）

### 样本B：day34-陪诊服务（正文叙事标准）
- 标题策略：对话开头（"我不认识路，怕挂错号"——她说这句话的时候65岁）
- 正文叙事角度：病人视角（从阿姨的困境切入，不是从陪诊者的成长切入）
- 配图4 Card 1 的prompt风格（"陪一位外地阿姨跑完全程省了2小时 / 把医生说的术语翻译成人话"）
- 群聊日常细节（"挂哪个科 / 几点去排队 / 空腹能不能喝水"）

## 标题多样性提醒

每篇轻创业的标题应该有不同的切入角度。以下是已使用的标题策略，新篇不要重复：

| 策略 | 示例 | 文档 |
|------|------|------|
| 观察+持续行动 | 那辆面包车在我们小区后门停了半年，风雨无阻 | 移动洗车 |
| 反常识开头 | 这话可能得罪人，但我观察了半年 | 夜市小吃 |
| 前后对比 | 他之前每天被PPT折磨，后来把折磨他的人变成了客户 | AIPPT |
| 对话开头 | "我不认识路，怕挂错号"——她说这句话的时候65岁 | 陪诊服务 |
| 具体场景 | 省人民医院一楼，有人靠陪人看病月入过万 | 陪诊服务v2（已废弃，太模板化） |

**避免重复的句式**：
- "那个XXX，我看了他四个月"
- "地点+有人靠XX赚钱"
- "那辆XXX，我观察了X个月"

## 配图1（5步路线图）密度参考

每张卡片结构：
```
Card标题 + icon
一句话原则（top line）
2-3个关键动作（middle section，用→或/连接，不要编号）
短句AI prompt（AI box，不超过2句，像自然聊天）
```

好的密度示例（来自后备箱咖啡Card 1）：
```
第1-3天 定方向 (compass icon)
top line: "选一个你愿意连续做30天的事"
middle: "验证方法: 去3个位置蹲点数人流 → 调研周边已有咖啡竞品 → 试做10杯请路人试喝"
AI box: "AI辅助: prompt「我想做XX轻创业，帮我分析赛道趋势、竞争程度、启动成本，给我7天验证计划」"
```

不要写成这样（太密）：
```
Card 1: ①去3个小区门口蹲点 数一个小时内经过的车流量
        ②调研周边洗车店的价格和服务(基础洗多少? 精洗多少?)
        ③买一套基础设备试洗2辆车(朋友的) 记录耗时和效果
        ④确定你能覆盖的范围(3公里? 5公里?)
        AI辅助: prompt「我想在[你的城市]做移动上门洗车，请帮我:
        ①分析本地市场需求和竞争情况 ②给出合理的定价建议
        ③列出第一周需要准备的工具清单，按优先级排序」
```

## 配图4（AI实操）prompt密度参考

用斜杠 `/` 分隔要点，像人在聊天框打字：

好的密度（后备箱咖啡风格）：
```
prompt: 「我在XX写字楼下卖咖啡 帮我写一条小红书笔记
/ 突出: 辞职做咖啡的第30天 / 每天早起占位置
/ 给老顾客做专属杯 / 语气真诚带点自嘲」
```

太密（说明书风格，避免）：
```
prompt: 「我是一名移动上门洗车师，今天给一辆黑色奥迪做了精洗+打蜡。
帮我写一篇小红书笔记，要求：
①标题要有吸引力 ②正文200字以内 ③语气真诚 突出上门洗车不用去排队的好处
④带3-5个相关标签」
```

## 配图2（赚钱逻辑）密度参考

- 收入公式：1行公式 + 4个注解（每个不超过1行）
- 收入阶梯：4级，每级一行（阶段 + 日收入 + 一句话描述）
- 真实成本：列出关键项 + 一个成本→利润的例子
- 底部对比：两组对比，一行一个

## 配图3（5大坑）密度参考

每个坑2行：
```
坑名 + 具体表现（1行，像说话，不要写报告）
避坑: 具体可操作建议（1行）
```

## Midjourney prompt中的关键词

每张图的末尾都加上：
- `Minimalist line art style, dark charcoal and warm gray`
- `[配色名] for AI boxes and key numbers`
- `Clean modern typography with generous spacing` （注意 generous spacing）
- `--ar 3:4 --v 6.0`
